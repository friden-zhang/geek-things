# esp32s3_devkit

USB-C powered ESP32-S3 development board built from reusable package blocks.

## Features

- ESP32-S3-WROOM-1 core with default I2C, SPI, UART, and native USB assignments
- USB-C sink-side input front-end via the upstream `atopile/usb-connectors` package
- BOOT, RESET, and USER buttons
- WS2812B-2020 addressable RGB status LED

## Build

```bash
ato build
```

Run the command from `hardware/boards/esp32s3_devkit/`.

## Notes

- The board-level `3V3` rail is generated by a concrete `AP2112K-3.3TRG1` LDO package.
- The board contract is the source of truth for default bus names and occupied GPIO assignments.

## Companion Files

- contract: `hardware/contracts/esp32s3_devkit.yaml`
- firmware target: `firmware/boards/esp32s3_devkit/`
