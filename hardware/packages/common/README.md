# Common Packages

Put reusable support circuits here when they do not fit cleanly into `interfaces/` or `power/`.

Examples include clock trees, LED indicators, MCU support glue, level shifting helpers, and sensor front-end fragments.

Current packages:

- `esp32s3_module_core.ato`: ESP32-S3-WROOM-1 core and default peripheral/breakout mapping
- `esp32s3_devkit_io.ato`: dev board buttons and addressable RGB status LED
